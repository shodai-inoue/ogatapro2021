# 骨格推定AIを使った手や体のポーズで家電を制御するシステム
## 動作環境
### ハードウェア
- Jetson Nano（AIと相性の良い開発ボード）
- Raspberry Pi Zero WH（Jetsonのサブボードとして利用）
- Webカメラ
- 赤外線LED
- 赤外線受信モジュール
### ソフトウェア
- Python3
- Media Pipe（骨格推定API）
